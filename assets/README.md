# Assets Folder

Place your marketing assets here:

- **Images** - Screenshots, photos, illustrations
- **Logos** - School logo, brand assets  
- **Icons** - Custom icons if needed
- **Videos** - Demo videos, testimonials

## Recommended Assets

1. **logo.png** - Your school logo (transparent background)
2. **hero-image.jpg** - Main hero section image
3. **dashboard-screenshot.png** - App screenshot
4. **features-*.png** - Feature screenshots
5. **favicon.ico** - Browser tab icon

## Image Optimization

- Use compressed images (use TinyPNG or similar)
- Recommended sizes:
  - Logo: 200x200px
  - Hero: 1200x800px
  - Screenshots: 1920x1080px
  - Icons: 64x64px

After adding images, update the `src` attributes in `index.html`!
